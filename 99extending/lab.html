Lab: Extending Titanium

You will build a simple "hello world" extension module for Titanium.

Specification: Our wiki includes guides on writing your own modules, which is a bit outdated (Studio now includes tools to perform many of the command-line operations described in that guide). The guides still provide useful architectural information and descriptions of the process that make it a worthwhile resource. In this lab, you will work through the steps of either the iOS or Android module development guide. Those guides walk you through a "Hello World" module project.

1. Our wiki includes guides on writing your own modules, which is a bit outdated (Studio now includes tools to perform many of the command-line operations described in that guide). The guides still provide useful architectural information and descriptions of the process that make it a worthwhile resource. In this lab, you will work through the steps of either the iOS or Android module development guide. Those guides walk you through a "Hello World" module project.
2. In Studio, choose File, New Titanium Module Project. Use 'HelloModule' as the project name. Enter an app ID, such as com.yourname.hellomod  and select the target platform you want to build for.
  1. For Android development, you'll need the Java Development tools. In the New Module Project dialog, click the Install JDT link and follow the steps on the resulting web page.
  2. Also for Android, confirm you have installed the necessary tools and components listed at Installing the Necessary Components
  3. Back in the New Module wizard, supply a version number and optionally the other information requested by the wizard.
3. Follow the steps in the module development guide for your chosen platform (links below). 
  1. For iOS modules, start at approximately this step: Open in Xcode
  2. For Android modules, start at approximately this step: Importing into Eclipse
4. In Studio, package your module by using the Package menu (in App Explorer, this is the box-shaped icon to the right of the Debug and Run buttons).
5. Create a Titanium mobile project to test your module
  1. Place the module into the modules/platform_name folder in your project
  2. Update the tiapp.xml as necessary (use the Add Module button in the Modules section of the Tiapp.xml editor window).
  3. Require() in your module in the app.js
6. Run your Titanium app in the simulator or emulator.
